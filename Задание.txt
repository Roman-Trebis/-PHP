Описание Задания

Задание: Разбить верстку на шаблоны. Стартовый код с версткой находится ниже на странице. Также ниже размещено видео с описанием итогового результата.



Цель задания: практика написания PHP кода. Исользование базовых возможностей языка: подключение файлов, переменные, константы, условия, массивы, циклы.



Базовое задание



1. Определить константы ROOT и HOST и использовать их для подключения файлов и для указания ссылок на сайте.


2. Выделить в шаблоны следующие части страниц:

Начальный код страницы с тегом head
Шапку (header)
Навигацию (nav)
Подвал и завершающий код страницы (footer)
Сайдбар
Отдельные шаблоны для виджетов в сайдбаре
3. На каждой странице выводить свой заголовок страницы (тег title).



4. В навигации выделять активный пункт, со страницей на которой находится пользователь в данный момент.



Расширенное задание



5. Сделать вывод постов на главной странице на основе массива с данными.

6. Сделать вывод данных в шаблонах виджетов (категории, популярные посты, фотографии) на основе массивов с данными.
